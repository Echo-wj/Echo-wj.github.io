一行命令生成二维码

一行命令生成一个二维码？

没错！前提是要python环境都安装好了，

然后再下载qrcode库

命令：pip install qrcode

ubutnu：apt install：sudo apt install python3-qucode

开始！

然后你的二维码就生成了！

再来点复杂的，搞点纠错级别这些东东。

那，再来点花里胡哨的！

先上图

这种有图片的还需要下载一个myqr库

参数如下

-v 指定二维码版本

-l 指定纠错级别

-c 可以显示彩色二维码

-n 指定二维码名称

-p 指定背景图片

既然能生成二维码，也肯定能识别二维码！
